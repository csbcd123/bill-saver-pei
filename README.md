# Bill Saver｜PEI 手机宽带账单免费体检

Minimal MVP for PEI internet and mobile bill analysis.

## Local development

```bash
npm install
cp .env.example .env.local
npm run dev
```

Open `http://localhost:3000`.

## Vercel

Set `OPENAI_API_KEY` in Vercel environment variables. `OPENAI_MODEL` is optional and defaults to `gpt-4.1-mini`.
